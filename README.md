# live-interaction

## 요구사항

1. 간단한 실시간 채팅 서비스
2. db에 채팅내용 저장해 활용

## Frontend
- Template engine : Pug
- CSS framework : Tailwind CSS

## Backend
- Web framework : Koa
- Live networking : koa-websocket
- Data base : MongoDB
